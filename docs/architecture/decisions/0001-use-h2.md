# ADR 0001: Use H2 Embedded Database

Date: 2026-01-05
Status: Accepted

## Context
Training environment has no Docker, no external database permitted.

## Decision
Use H2 embedded in PostgreSQL-compatibility mode. Migrations remain portable.

## Rationale
- Zero install
- Same JVM as Spring Boot
- Flyway works identically
- Production will swap to real PostgreSQL without migration changes

## Consequences
- No PostgreSQL-specific types (ARRAY, JSONB, UUID type)
- UUIDs stored as VARCHAR(36)
- Tests fast: no container startup
